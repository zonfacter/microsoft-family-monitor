# microsoft-family-monitor

Microsoft Family Bildschirmzeit-Controller.

Eine inoffizielle lokale WebAPI zur Verwaltung und Überwachung der
Bildschirmzeit mehrerer Kinderkonten
mit Microsoft Family Safety.

Das Projekt bietet ein modernes, benutzerfreundliches Dashboard und ermöglicht die Steuerung der Bildschirmzeit direkt vom lokalen PC.
